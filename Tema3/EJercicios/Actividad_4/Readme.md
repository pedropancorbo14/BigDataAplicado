# Estructura de carpetas

## Datos_crudos (Bronce)

Archivos crudos de los datos antes de ser procesados

## Datos_procesados (Plata)

Archivos filtrados con los datos ya preparados

## Analizados (Oro)

Archivos visuales y dinámicos de los datos ya preparados

## Años y evaluaciones

Dentro de cada tipo habrá carpetas de años (2025,2026...), y dentro de esos años carpetas con Evaluaciones
